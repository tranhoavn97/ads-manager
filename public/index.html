Tạo hàng trăm quảng cáo từ một file Excel.

Tải lên file Excel hoặc Google Sheet, hệ thống tự đọc Page ID, Post ID, kiểm tra quyền từng dòng rồi tạo campaign, nhóm quảng cáo và quảng cáo qua Meta Marketing API — có màn hình kiểm tra trước khi chạy.

Đăng nhập với Facebook for Business

Quyền yêu cầu: ads_management, ads_read, pages_show_list, pages_read_engagement, pages_manage_ads.

Lấy token từ Graph API Explorer hoặc System User. Token cần đủ các quyền ở trên. Token được lưu trong cookie phiên (8 giờ), không hiện ra mã nguồn trang.
